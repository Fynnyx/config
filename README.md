# .dot Files Repository


## Prerequisites

Install Discord to send secrets across devices:
```bash
snap install discord
```


Install Brave and add Sync Chain:

[Brave Download Page](https://brave.com/linux/)


Install git
```bash
sudo apt-get install git
```


Generate SSH keys for GitHub if you don't have them already:
```bash
ssh-keygen
```

Add them to the GitHub account.

## Installation
Clone this repository to your home directory:
```bash
git clone git@github.com:Fynnyx/config.git
cd config
bash setup.sh
```

## Further Program Installations
Install VSCode:
```bash
sudo snap install --classic code
```
